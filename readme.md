---

### 1. ระบบสมาชิก (Authentication & Users)

จัดการเรื่อง Login, Register และข้อมูลส่วนตัว

| Method    | Path                 | คำอธิบาย                                               |
| :-------- | :------------------- | :----------------------------------------------------- |
| **POST**  | `/api/auth/register` | สมัครสมาชิกใหม่ (ลงตาราง USERS)                        |
| **POST**  | `/api/auth/login`    | เข้าสู่ระบบ (รับ Token)                                |
| **GET**   | `/api/users/profile` | ดูข้อมูลตัวเอง (ชื่อ, วันเกิด เพื่อเอาไปคำนวณหินนำโชค) |
| **PATCH** | `/api/users/profile` | แก้ไขข้อมูลส่วนตัว (เช่น เปลี่ยน nickname หรือวันเกิด) |

---

### 2. ระบบสินค้าและการค้นหา (Stones & Discovery)

ดึงข้อมูลหิน และระบบแนะนำหินตามวันเกิด (Birth Match)

| Method   | Path                    | คำอธิบาย                                               |
| :------- | :---------------------- | :----------------------------------------------------- |
| **GET**  | `/api/stones`           | ดูรายการหินทั้งหมด (มี Filter ตาม category ได้)        |
| **GET**  | `/api/stones/:id`       | ดูรายละเอียดหินเม็ดนั้นๆ (พร้อม power_stats)           |
| **GET**  | `/api/stones/recommend` | **(Highlight)** ระบบคำนวณหินที่เหมาะกับวันเกิดผู้ใช้   |
| **POST** | `/api/stones/search`    | ค้นหาหินแบบ Advanced (ใช้ search_vector ค้นหาความหมาย) |

---

### 3. ระบบสั่งซื้อและชำระเงิน (Orders & Payments)

จัดการตะกร้าสินค้าและการจ่ายเงิน

| Method   | Path                | คำอธิบาย                                          |
| :------- | :------------------ | :------------------------------------------------ |
| **POST** | `/api/orders`       | สร้าง Order (ตัดสต็อกใน STONES และลง ORDER_ITEMS) |
| **GET**  | `/api/orders`       | ดูประวัติการสั่งซื้อทั้งหมดของ User               |
| **GET**  | `/api/orders/:id`   | ดูรายละเอียดของ Order นั้นๆ (สั่งอะไรไปบ้าง)      |
| **POST** | `/api/payments`     | แจ้งชำระเงิน (แนบ payment_slip และอัปเดตสถานะ)    |
| **GET**  | `/api/payments/:id` | เช็คสถานะการชำระเงิน                              |

---

### 4. ระบบ AI Chatbot (AI Assistant)

ส่วนติดต่อสื่อสารกับ AI เพื่อขอคำแนะนำ

| Method     | Path                              | คำอธิบาย                                              |
| :--------- | :-------------------------------- | :---------------------------------------------------- |
| **POST**   | `/api/chat/sessions`              | เริ่มเปิดห้องแชทใหม่                                  |
| **GET**    | `/api/chat/sessions`              | ดูรายการห้องแชทเก่าๆ                                  |
| **POST**   | `/api/chat/messages`              | **(Core)** ส่งคำถามให้ AI (ต้องส่ง session_id ไปด้วย) |
| **GET**    | `/api/chat/sessions/:id/messages` | ดึงประวัติการคุยในห้องนั้นมาโชว์                      |
| **DELETE** | `/api/chat/sessions/:id`          | ลบประวัติการคุยที่ไม่ต้องการ                          |

---

### 5. ระบบบันทึกการอ่าน (Reading History)

| Method   | Path                   | คำอธิบาย                                               |
| :------- | :--------------------- | :----------------------------------------------------- |
| **POST** | `/api/history/reading` | บันทึกเมื่อ User เข้าไปอ่านรายละเอียดหินหรือดูดวง      |
| **GET**  | `/api/history/reading` | ดึงประวัติที่เคยอ่าน (ให้ AI ใช้แนะนำหินที่ User สนใจ) |
